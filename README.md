# DWAFM

Dynamic Weighted Graph Structure Embedding Integrated with Attention and Frequency-Domain MLPs for Traffic Forecasting


## Environment
- python 3.11 
- torch 2.3.1  + cu121
- numpy 1.26.4
- SEED = 1
